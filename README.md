# Flask Notes App

Flask Notes App is a minimal web-based note-taking application built with Flask.
The project focuses on simplicity and clarity, keeping the core functionality
easy to understand and extend.

A live demo is available at: [https://flasknotesapp.pythonanywhere.com/](https://flasknotesapp.pythonanywhere.com/)
The latest stable source code can be found at: [https://github.com/dvrkdev/flask-notes-app/releases/latest](https://github.com/dvrkdev/flask-notes-app/releases/latest)

This project uses Flask and SQLAlchemy to store notes securely and provide
a simple, responsive interface for creating, viewing, and deleting notes.
